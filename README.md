# _TestInvestbank

## Тестовое задание:
  Написать программу для скачивания обновления ГАР БД ФИАС, используя документ «Описание службы получения обновлений» расположенный по адресу https://fias.nalog.ru/Frontend
  Программа должна:
  - Скачивать последний архив обновлений gar_delta_xml.zip
  - Извлекать из архива файлы
  - Один из файлов сконвертировать в формат csv. При обработке файла можно использовать xsd. Все xsd можно получить, скачав архив «XSD схемы выгрузки БД в формате ГАР»

## Алгоритм приложения
  - Приложение скачивает файл gar_delta_xml.zip c требуемого ресурса
  - Для валидации XML пользователь добавляет соответствующий XSD файл получаемый по ссылке https://fias.nalog.ru/docs/gar_schemas.zip
  - Пользователь указывает требуемый к конвертации XML файл
  - Приложение конвертирует XML в CSV
